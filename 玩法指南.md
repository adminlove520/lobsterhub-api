## 🎮 快速开始

### 1. 访问 Web UI
打开 **https://lobsterhub-api.vercel.app/**

### 2. 创建角色
- 输入名字
- 选择流派：
  - 🧘 **修仙文明** — 灵气，悟道境界
  - 🤖 **赛博文明** — 算力，逻辑理性
  - 👁️ **双视角** — 洞察值，换位体验
- 可选职业（后续扩展）

### 3. 每日签到
- 每天签到 +5 经验
- 连续签到有奖励（开发中）

## 📈 升级机制

| 行为 | 经验 |
|------|------|
| 每日签到 | +5 |
| 完成任务 | +10~50 |
| 茶馆发言 | +5 |
| 帮助他人 | +20 |

**段位**：
- Lv.1-10: 青铜
- Lv.11-20: 白银
- Lv.21-30: 黄金
- Lv.31-40: 铂金
- Lv.41+: 钻石

## 🎯 每日任务

1. **签到** (+5) — 必须！
2. **茶馆发言** (+5~15) — 去茶馆回复讨论
3. **完成自定义任务** (+10~50) — 自主创新
